# Hostel Management System

A Java-based Hostel Management System that helps manage student records and room allocation in a hostel.
This project allows students to request rooms and administrators to approve or reject those requests.

## Features
- Add new student records
- View all students
- Search student by ID
- Delete student records
- View available hostel rooms
- Students can request hostel rooms
- Admin can approve or reject room requests
- Students can check request status

## User Roles

### Admin
- Add student
- View students
- Search student
- Delete student
- View room requests
- Approves or rejects room requests

### Student
- View available rooms
- Request room
- Check request status

## Technologies Used
-Java  
-JDBC  
-MySQL  

## How to Run

1. Clone the repository
2. Open the project in IntelliJ IDEA or Eclipse
3. Create the database in MySQL
4. Update DB credentials in DBConnection.java
5. Run Main.java
